靜態網站生成工具

2016年9月11日

定期查看一下 StaticGen - Top Open-Source Static Site Generators 上的排行。

Jekyll 毫無疑問是遙遙領先的王者，我猜是因為 GitHub Pages 的預設支援，當然也有可能是使用 ROR 的開發者也都理所當然使用相同語言的工具。

GitBook 就完全讓人跌破眼鏡。它並非嚴謹異議下的靜態「網站」生成，而是製作「線上電子書」的工具，不少人用它撰寫有著書籍架構的說明文件。

Hexo 好像是台灣人寫的工具呢！純 JavaScript，彈性頗大，也有方便的終端機指令工具與部署套件。目前使用中。

後面還有一個希望多摸摸的工具： ROOTS。
